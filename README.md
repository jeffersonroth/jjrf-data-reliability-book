<!-- GETTING STARTED -->
## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/jeffersonroth/jjrf-data-reliability-book.git
   ```
2. Install cargo packages
   ```sh
   cargo install mdbook
   cargo install mdbook-linkcheck
   cargo install mdbook-man
   ```
